GoFish - A Social Media search engine

coding, design

    • Screenshot od GoFish search results
    • GoFish logo
    • GoFish is a prototype for a Social Media search engine searching Twitter, Delicious and Digg. It's built in PHP and uses SimplePie to pull RSS feeds for a search term from APIs of one or more (depending on the user's selection) Social Media sites.
    • The prototype was designed and built as a project for a course in knowledge management. It is intended as a tool for tracking topics across different Social Media platforms without having to visit every single site separately. In addition to the 15 most recent search results from each site, a bar graph (dubbed discussion meter) is generated, offering a visual comparison of the number of search results from each platform.
    • For prototyping purposes, only three Social Media platforms were included, but any site offering a RSS feed fo search results can be integrated.
    • The logo was designed to illustrate the GoFish metaphor of fishing for information, not in the ocean of the entire Web, but in whichever "bowl" the user chooses. It was created in Photoshop with a simple icon-style look emulating the site's clean, modern Web 2.0 look.
